This post walks through the goals behind this portfolio: fast iteration, accessible defaults, and content that is easy to update without touching layout code.

## Why MDX

MDX lets you mix Markdown with React components when you need more than prose. For a personal site, that balance keeps posts portable while leaving room for embeds or callouts later.

## Performance checklist

- Prefer `next/image` for every raster asset so you get automatic sizing and modern formats.
- Keep interactive widgets behind dynamic imports when they are not needed for first paint.
- Reach for server components for data fetching and static rendering whenever possible.
